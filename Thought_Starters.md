# Thought Starters

Ideias iniciais para ajudar sua equipe a sair do branco e transformar o tempo curto do hackathon em uma demo funcional. Todas as sugestões abaixo já assumem o uso de **Agora ConvoAI + Agora RTC**, que são obrigatórios para os projetos elegíveis.

---



## Devin por voz

Interface de voz usando Agora ConvoAI que comanda o Devin via Custom LLM proxy. Usuário fala 'cria uma função que X' → ConvoAI → Devin → resposta por voz. Qualifica pra trilha Agora.



## Atendimento ao cliente autônomo

Voice agent com ConvoAI que resolve tickets de suporte escrevendo código via Devin quando necessário (hotfix ao vivo com cliente na linha).



## Pair programmer real-time

Sessão Windsurf com Agora RTC streaming (voz + tela) para colaboração remota síncrona entre devs e Devin.



## Code review por voz

Engenheiro conversa com um agent que leu o PR e explica mudanças via TTS; agent acionado pelo Devin no backend.



## Voice-first dashboard de observabilidade

Agent que responde 'como está a latência da API?' buscando métricas e narrando resultado via voz (ConvoAI + Oracle OCI metrics).
